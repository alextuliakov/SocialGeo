# Описание
Социальная сеть для путешественников с привязкой постов к геолокации.
Код загружен на GitHub в 2026 году с моего аккаунта Git-Lab Школы Программистов https://gitlab.informatics.ru/

Это итоговый выпускной командный проект 2020 года. Защита проекта проходила на базе компании Яндекс в Москве.
Разработан командой из "Школы Программистов" - "Клуб Анонимных Стесняш" :))

В лично мои обязанности входила разработка профиля пользователей и всего, что с этим связано.
